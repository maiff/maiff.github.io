Hexo博客搭建之路

发表于 2016-02-25   |   分类于 技术应用   |  

为什么要换博客

首先我最开始用的是博客园的博客，自由度很高，可以改css可以用js，都很完善，但是我实在忍受不了，博客园在我眼里真的很丑。然后我转去wordpress，然后空间对于我这种月光族实在太贵了，不过后来发现有阿里云的学生云翼计划每个月只要九块，可是问题还是出现了，我的学生证不能办，只能用我女朋友的，可是她马上就要毕业了就不能用了，我就急于找一个好的解决方案，由于我不是苹果的电脑，只能用hexo

hexo安装

我开始的时候就安装了NodeJS，还有git，就没有在这上面遇到问题。
然后就是hexo的安装，我在网上找到好几个版本，大概就是在不同的地方用git命令，最后都是要到自己想安装hexo的位置git bash。
然后执行命令$ npm install -g hexo，安装hexo。

部署hexo

$ hexo inithexo会自动在目标文件夹建立网站所需要的所有文件。
然后用hexo g和hexo s就可以用localhost:4000在本地看到自己的博客，至此我还没有碰到问题。

连接github

本地的博客只能自己看，这样就没意思了，我们需要大家一起交流，由于hexo是静态博客，可以放在github上，这样就需要连接github，连接github肯定要有github的账号，我之前就有。然后建一个 repositorie，以yourname.github.io命名，我不知道为什么要以这个命名？开启GitHub Pages功能。然后检查本机的ssh，$ cd ~/. ssh，如果提示：No such file or directory 说明你是第一次使用git。然后我本来就有，直接看到了，如果没有就要用命令$ ssh-keygen -t rsa -C "邮件地址@youremail.com"然后会提示一大堆，如果你不是很熟，就一路确认一下，然后就会生成一大串ssh

添加SSH Key到GitHub

在本机设置SSH Key之后，需要添加到GitHub上，以完成SSH链接的设置。

  • 1、打开本地C:\Documents and Settings\Administrator.ssh\id_rsa.pub文件。此文件里面内容为刚才生成人密钥。如果看不到这个文件，你需要设置显示隐藏文件。准确的复制这个文件的内容，才能保证设置的成功。
  • 登陆github系统。点击右上角的 Account Settings—>SSH Public keys —> add another public keys
  • 3、把你本地生成的密钥复制到里面（key文本框中）， 点击 add key 就ok了

    ###测试 ###
    可以输入下面的命令，看看设置是否成功，git@github.com的部分不要修改：$ ssh -T git@github.com
    然后这里我遇到问题还纳闷半天，我以为这里的邮箱要填自己的，害我查了半天……这里应该是github的服务器。

    设置用户信息

    现在你已经可以通过SSH链接到GitHub了，还有一些个人信息需要完善的。

Git会根据用户的名字和邮箱来记录提交。GitHub也是用这些信息来做权限的处理，输入下面的代码进行个人信息的设置，把名称和邮箱替换成你自己的，名字必须是你的真名，而不是GitHub的昵称。

$ git config --global user.name "cnfeat"//用户名
$ git config --global user.email  "cnfeat@gmail.com"//填写自己的邮箱

然后在hexo的配置文件中修改

# Deployment
## Docs: http://zespia.tw/hexo/docs/deployment.html
deploy:
  type: git
  repository: https://github.com/yourname/yourname.github.io.git
  branch: master

每次就能用hexo d提交到guthub上，不过我发现每次都要输github的账号密码，有点烦，最后谷歌到设置环境变量就好了。教程
至此每次修改更新就很方便了。

更改域名

之前就有这样的经验，申请个域名，改解析地址CNAME，然后github里pull一个CNAME，没问题到这，后来我发现，遇到问题了，每次hexo d之后我的CNAME就没了，后来谷歌到，只要把CNAME文件放到source文件中，每次就会pull上。问题解决。

使用多说

很简单在hexo配置文件中加一个duoshuo_shortname: your-duoshuo-name；仅限我这个next主题。不过以后要仔细看教程。
不过每个主题不一样，自行谷歌吧。

更改站点ico

很简单，多一个图标favicon.ico放到source中就好。
